Installation

Importan! libmodbus4 work only in Linux и OSX

  1. If you not have interpritator Ruby-1.9.x download and read instruction of installation on official site

  2. Installation libmodbus from source

    $ wget http://launchpad.net/libmodbus/trunk/2.0.3/+download/libmodbus-2.0.3.tar.gz

    $ tar -xf libmodbus-2.0.3.tar.gz

    $ cd libmodbus-2.0.3

    $ ./configure --prefix=/usr

    $ make

    $ sudo make install

  3. Installation libmodbus4r Simplest installation is use rubygems

    $ sudo gem install libmodbus4r

    or installation from source with command:

    $ sudo rake install
